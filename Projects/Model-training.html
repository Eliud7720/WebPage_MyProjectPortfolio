About this project

Once all features for all proteins were prepared (remembering that there were three feature sets per protein: RDKit molecular descriptors, Mordred descriptors, and a 2048-bit molecular fingerprint), machine learning models were trained to later make predictions on a completely new and unseen dataset. This involved selecting the machine learning models to be used and performing their training according to the guidelines established by MLOps.

Objective

Train and evaluate all machine learning and deep learning models for each protein using their three sets of molecular descriptors.

Development

Data collection

The three sets of molecular descriptors (RDKit, Mordred, and the molecular fingerprint) were obtained from the previous feature selection step for each protein. Therefore, each protein has three sets: one with 80 most relevant RDKit descriptors, another with 80 most important Mordred descriptors, and finally a 2048-bit fingerprint representing the entire molecule.

Selected models

For the purposes of this research, the following machine learning models were trained: Logistic Regression, Support Vector Machine (SVM), Random Forest, and XGBoost using the Scikit-learn library in Python. Additionally, a multilayer perceptron (MLP) architecture was designed and trained using the TensorFlow library for Python.

Models training

Conclusions

  • All proteins (EGFR, MAPK1, SRC, STAT3) were modeled using three feature sets: RDKit descriptors, Mordred descriptors, and a 2048-bit ECFP4 fingerprint.
  • Five types of models were trained: Logistic Regression, SVM, Random Forest, XGBoost, and a Multilayer Perceptron (MLP) for deep learning.
  • Training used five-fold cross-validation to determine the best hyperparameters, which were then applied to evaluate models on test datasets.
  • Models were evaluated using Accuracy, Precision, F1-score, and Recall, with color-coding highlighting the best-performing models and descriptors.
  • The ECFP4 fingerprint was the most relevant descriptor overall, consistently associated with the highest-performing models across all proteins.
