Web Scraping of ProTox to Automate Toxicity Prediction of a Compound Set

About this project

After the drug discovery phase (the stage in which the researcher determines compounds that inhibit or potentially inhibit the protein of interest), an in silico prediction is carried out to predict potential toxicity in order to perform another screening and only retain molecules that potentially do not have toxic effects on humans.

ProTox is a website that includes a wide variety of machine learning tools to predict the toxicity of various aspects; however, it only allows the toxicity of a single molecule to be calculated at a time.

Therefore, for this project, I was hired to develop a script that would allow ProTox to automatically work with a set of molecules and carry out predictions.

Objective

Develop a Python script that automatically performs predictions for a set of compounds on the ProTox page and stores the results in CSV format, respecting the page rules at all times.

Development

About Protox

ProTox is an online platform that provides a variety of machine learning tools to predict the potential toxicity of chemical compounds. It helps researchers assess whether molecules might have harmful effects on humans. However, the standard ProTox interface only allows predictions for one molecule at a time, which can be limiting for large datasets.

The following screenshots show the ProTox page, specifically the section where all stages of the prediction process are configured. Here, the SMILES of the compound of interest is entered, and all relevant prediction options are selected for analysis.

Script Development

Due to the intensive work required to analyze, for example, 60 molecules manually and compile all the results into a single DataFrame, I developed a script that automatically reads data from CSV files, performs the predictions on the ProTox website, and finally combines all results into a single CSV file.

For the development of this script, the Python Selenium library was used, along with pandas and numpy to work with DataFrames. The script is designed to load a CSV file that contains a column named SMILES with the compounds' SMILES strings. It then automatically opens the Mozilla browser, inputs each SMILES, selects all the required prediction options, and finally saves all the results into a single CSV file.

Example of use

This script has been used countless times. In my case, I analyzed up to 200 different compounds to determine their potential polypharmacological activity‚Äîthat is, whether a compound could inhibit more than one protein, which is undesirable in drug design as it indicates potential side effects and toxicological risks.

After obtaining the results in the format shown previously, I loaded the data using pandas in Python and created a new column representing the number of proteins potentially inhibited by each compound. I then grouped all compounds by the number of proteins they inhibited and identified the specific proteins for compounds inhibiting 1, 2, and more proteins. Finally, I generated a histogram showing these distributions.

Additionally, for each group, I identified the maximum common substructure among the compounds, which is displayed at the top of the following histogram.

As shown in the previous histogram, the vast majority of compounds exhibited only 1 to 2 proteins as polypharmacological activity. The top section shows that molecules with this low activity were similar to staurosporine and other heterocyclic molecules. However, at least two heterocyclic molecules showed potential activity against 5 or more proteins.

The proteins potentially inhibited are displayed at the top, and the ‚Äú+‚Äù sign indicates that the compound inhibits this protein in addition to the previous ones, showing a trend where certain compounds inhibited the same proteins while also affecting additional ones.

Conclusions

  • The project automated toxicity predictions on ProTox, which normally allows only one molecule at a time.
  • The script significantly reduced manual workload, allowing analysis of large datasets, up to 200 compounds in my case.
  • Results were analyzed to determine polypharmacological activity, i.e., the number of proteins each compound could potentially inhibit, identifying potential side effects and toxicological risks.
